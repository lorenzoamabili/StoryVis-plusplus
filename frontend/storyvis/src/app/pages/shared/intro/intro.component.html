With this study, we would like to test whether the StoryVis tool and some of its features are effective to explore and communicate medical imaging data.
Before the start of the experiment, a short tutorial about how to use the application is shown and you will have practice time to become familiar with the tool.
Once ready, you can start performing the tasks and you have unlimited time to complete them. As a first task, we ask you to perform an image assessment of the given data based on clinical questions. Then, we ask you to create a text report/visual data story to communicate the results of the image assessment. Feel free to go back to extend the image assessment if needed.
Finally, we ask you to complete a short questionnaire.
When you are done, you can end the experiment at any time.
Notice that we are not testing you, we are testing our techniques!
