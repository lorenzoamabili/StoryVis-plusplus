# StoryVis for running user studies

## Backend 

'cd backend'
'yarn install' 
'yarn start'


## Frontend

'cd frontend'
'yarn install'
'cd provenance-core && yarn build'
'cd ami && yarn build:ami'
'cd provenance-tree-visualization-grouping && yarn build'
'cd slide-deck-visualization && yarn build'
'cd storyvis && yarn start'
